Pinterest API Usage

The Corner uses the Pinterest API to post pins to authorised Pinterest boards. We request the following OAuth scopes: boards:read, pins:read, pins:write, user_accounts:read. We do not store your Pinterest credentials. Access tokens are stored securely and used solely to post curated product pins on your behalf. You may revoke access at any time via your Pinterest security settings.
